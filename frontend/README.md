# Fact Checker - Web App

## Stack

- Typescript
- React
- Redux-toolkit
- Material UI
- Emotion CSS
- Axios
- Vite


## Features

* Check URL/text for trustworthiness
* URL check statistics
* Whitelist/Admin panel


## Installation

To run the server, you will need to have Node JS installed.
After that, run this command:

```
$ npm install
```

## Development

```
$ npm run dev
```

## Build

```
$ npm run build
```

## Start

```
$ npm run preview
```
